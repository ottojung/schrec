# Usage

See example programs in `example/`.

To run them, do

```
 make
```

followed by

```
 dist/schrec example/<NAME-OF-THE-EXAMPLE>.scm
```

Use `--results first` option if only want to get the first result.
Sometimes, there are just too many of them.

# Dependencies

- GNU Guile >= 2
